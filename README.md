# ArabicDataset
 About 7,000 Arabic sentences
* Economy (اقتصاد)
* Political (سياسة)
* Sport (رياضة)
* Culture (ثقافة)
* General (جمل عامة بدون موضوع مُحدد)
# Why is this dataset different from others?
* Higher accuracy and clarity.
* Multiple manual reviews.
